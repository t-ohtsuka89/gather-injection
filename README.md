# Gather Injection

## インストール方法

1. このリポジトリをクローンします：
   ```
   git clone https://github.com/t-ohtsuka89/gather-injection.git
   ```

2. プロジェクトディレクトリに移動します：
   ```
   cd gather-injection
   ```

3. 必要な依存関係をインストールします：
   ```
   pip install .
   ```

4. プロジェクトを実行します：
   ```
   gather-injection -s "{JavaScriptのファイルパス（複数指定可）}"
   ```
